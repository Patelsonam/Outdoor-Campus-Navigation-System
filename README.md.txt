# Outdoor Campus Navigation System

The Outdoor Campus Navigation System is a web-based application developed to
assist students, faculty members, and visitors in navigating large university
campuses. The system provides an easy-to-use interface that helps users locate
buildings, departments, and important outdoor locations within the campus.

This project aims to reduce confusion for new students and visitors by offering
clear navigation guidance and an interactive campus layout.
